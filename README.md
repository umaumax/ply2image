# ply2image

## how to use
download ply file
``` bash
wget http://www.cc.gatech.edu/data_files/large_models/bunny.ply.gz
gunzip bunny.ply.gz
```

``` bash
./ply2image.py bunny.ply

# see ./output_animation.gif
# see ./output_images/image_XX.png
```
